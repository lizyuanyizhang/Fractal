# 页面无法正常运作 - 故障排查指南

## 🔍 检查步骤

### 1. 打开浏览器开发者工具

1. 按 `F12` 或 `Cmd+Option+I`（Mac）
2. 查看 **Console** 标签页
3. 查看是否有错误信息

### 2. 检查各个页面的问题

#### Prism 页面问题

**可能原因：**
- React Flow CSS 未加载
- 节点位置计算错误
- React Flow 初始化失败

**检查：**
- 控制台是否显示 "✅ ThePrism 组件已加载"
- 是否有 React Flow 相关错误
- 页面是否完全空白

**解决方案：**
如果页面完全空白，可能是 React Flow 初始化问题。

#### Gravity 页面问题

**可能原因：**
- 容器尺寸未正确计算
- 任务气泡未显示
- 拖拽功能失效

**检查：**
- 控制台是否显示 "✅ InterestGravity 组件已加载"
- 是否显示 "加载中..."
- 是否有任务气泡显示

**解决方案：**
如果一直显示"加载中"，可能是容器尺寸计算问题。

#### Tunnel 页面问题

**可能原因：**
- 全屏模式覆盖了导航栏
- 计时器未启动
- 粒子特效未加载

**检查：**
- 控制台是否显示 "✅ FocusTunnel 组件已加载"
- 是否显示任务标题和计时器
- 导航栏是否可见

## 🛠️ 常见问题修复

### 问题 1：页面完全空白

**可能原因：** JavaScript 错误导致组件无法渲染

**解决方案：**
1. 查看浏览器控制台的错误信息
2. 检查是否有红色错误
3. 截图错误信息

### 问题 2：Prism 页面看不到节点

**可能原因：** React Flow 未正确初始化

**临时解决方案：**
- 刷新页面
- 检查网络连接（React Flow 可能需要加载资源）

### 问题 3：Gravity 页面看不到气泡

**可能原因：** 容器尺寸为 0

**检查：**
- 查看控制台是否有 "加载中..." 提示
- 检查容器是否正确初始化

### 问题 4：Tunnel 页面无法退出

**可能原因：** 导航功能未正确传递

**解决方案：**
- 点击导航栏的其他按钮应该可以切换
- 如果不行，刷新页面
