# Fractal

一个专为 INTP 人格类型设计的非线性待办事项应用。

## 设计理念

- **极简主义**: 去除一切不必要的干扰
- **赛博朋克风格**: 深色背景 + 霓虹绿/紫色强调
- **非线性交互**: 使用思维导图式任务拆解
- **丝滑动画**: Framer Motion 驱动的微交互

## 技术栈

- React + Vite (TypeScript)
- Tailwind CSS
- Framer Motion
- React Flow
- Lucide React

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
